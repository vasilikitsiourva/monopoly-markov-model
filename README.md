# Monopoly Markov Model

## Overview
This project models player movement on the Monopoly board using a Markov chain approach. It computes the probability of landing on each square based on dice rolls and board mechanics, and provides an interactive visualization.

## Features
- Transition matrix for Monopoly board
- Steady-state probability computation
- Simulation of position after n rolls
- Interactive GUI using Pygame

## Technologies
- Python
- NumPy
- SciPy
- Matplotlib
- Pygame
